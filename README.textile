Make your taurballs cower in fear.

h2. Usage

pre. Usage: cower [options] <operation> PACKAGE [PACKAGE2..]

pre. Operations:
  -d, --download          download PACKAGE(s)
                            pass -dd to download AUR dependencies
  -i, --info              show info for PACKAGE(s)
  -s, --search            search for PACKAGE(s)
  -u, --update            check for updates against AUR. If the 
                            --download flag is passed as well,
                            fetch each available update.

pre. General options:
  -c, --color             use colored output
  -f, --force             overwrite existing files when downloading
  -q, --quiet             output less to stdout
  -t DIR, --target=DIR    specify an alternate download directory
  -v, --verbose           be more verbose


h2. Dependencies:

* jansson (JSON handling)
* curl (fetching)
* libarchive (taurball extraction)
* pacman (pacman functions and data structures)
